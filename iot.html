Company Discription:

Blee is a wearable band for people suffering from deafness that converts emergency sound alerts to vibrations and light. Deaf people miss out on many sound notifications such as doorbell, fire alarm, traffic honks, baby crying etc. User can record and save any sound from the environment. Whenever the same sound is picked by the band, it vibrates and notifies user about the sound. E.g. A person can save his/her name in the device by recording it. Whenever someone calls that name out, the user is informed about it. The device has multiple vibrators and lights that also indicated the direction of the sound source. Each sound is also associated with particular vibration pattern and colour code so that the user can distinguish between different sound alerts.

http://www.getblee.com/
